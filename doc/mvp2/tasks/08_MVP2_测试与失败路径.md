# MVP-2 第八阶段：测试与失败路径

## 1. 工作目标

为 MVP-2 增加 package-local 测试和跨 package 集成测试，覆盖 MoveIt 适配层、
PlanningScene、attach/detach 和任务编排失败路径。

## 2. 推荐测试

```text
moveit_arm_control package test
PlanningScene object test
AttachObject / DetachObject test
assembly_task MVP-2 integration test
launch smoke test
```

## 3. 失败路径

至少覆盖：

```text
MoveIt server 不可用
named target 不存在
规划失败
执行超时
object 不存在
重复 attach
未 attach 时 detach
terminal operation 非 PLACE
```

## 4. 验收标准

```text
colcon test 通过
成功路径和主要失败路径均有断言
失败时 TaskState 进入 FAILED
错误码和 message 可定位原因
```
