# MVP-2 总体任务说明：RViz / MoveIt 单 FR3 Pick-and-Place

## 1. 工作目标

MVP-2 的目标是把 MVP-1 中的 Fake 机械臂底层能力替换为 FR3 MoveIt / RViz
仿真能力，同时尽量保持上层任务接口、状态机语义和错误处理方式稳定。

本阶段仍然是单臂、夹爪化末端和固定 Pick-and-Place，不接真实 FR3 真机，
不做 RH56DFX 全关节真实控制。

## 2. 总体链路

```text
StartTask(pick_and_place_mvp)
-> ResetScene
-> ControlHand(open / preshape / close / release)
-> MoveArm(right_arm, named target)
-> AttachObject / DetachObject
-> ExecuteTerminalOperation(PLACE)
-> TaskState
```

MVP-2 的核心变化：

```text
fake_arm_control
-> moveit_arm_control
```

`assembly_task` 仍应调用稳定的 `MoveArm.action`，不直接依赖 MoveIt 或
厂商 API。

