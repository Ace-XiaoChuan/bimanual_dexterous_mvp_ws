# Post-MVP 1 人工示教与真机安全基线

本目录保存 Post-MVP 1 的现场参考材料，用于记录已有真机工位和人工示教装配的
起点。它们不是本仓库软件自主运行的验收证据，也不表示已完成自主装配。

## 材料

| 文件 | 内容 | 作用 |
| --- | --- | --- |
| `human_demonstration_workcell.jpg` | 双臂真机工位照片 | 记录现场工位参考 |
| `human_demonstration_assembly.mp4` | 约 6 秒的人工示教装配视频 | 记录人工示教参考动作 |

## 使用边界

- 公开视频、演示或对外提交前，应确认拍摄许可、人员隐私和公司设备信息的公开范围。
- 后续若将人工示教轨迹转化为项目输入，应另行记录数据来源、坐标系、采样方式和
  可复现的处理过程。
- Post-MVP 1 的正式完成仍需独立的安全检查、驱动启动和低速运动验收记录。
